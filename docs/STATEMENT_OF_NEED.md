# Statement of Need

GeoPoly is an educational and maker-oriented polyhedra laboratory designed for classroom use, fabrication workshops and computational geometry demonstrations. It combines polyhedra catalogs, Euler/topological validation, formal Johnson solids, printable nets, fixed-scale tiling, batch material optimization and multiple export formats in an offline Python application.

The practical need is to bridge a gap between mathematical visualization tools and fabrication-ready classroom material. GeoPoly focuses on verified printable nets, glue-tab labeling, A4/Letter page workflows, headless export tests and batch optimization for preparing activities for a whole class or makerspace session.

The algorithms used are mostly established computational-geometry and packing heuristics; the contribution is the integrated, reproducible, tested and educationally oriented software workflow.
